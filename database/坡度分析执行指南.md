# 坡度分析数据导入和计算指南

## ⚠️ 当前状态

## 🚀 启动PostgreSQL服务

### 方法一：通过Windows服务管理器

1. 按 `Win + R` 打开运行对话框
2. 输入 `services.msc` 并回车
3. 找到 `postgresql-x64-18` 服务
4. 右键点击 → 启动

### 方法二：通过命令行

```powershell
# 以管理员身份运行PowerShell
Start-Service postgresql-x64-18
```

### 方法三：通过pgAdmin

如果安装了pgAdmin，可以通过图形界面启动服务。

## 📁 执行坡度分析步骤

### 步骤1：添加坡度字段

启动PostgreSQL后，执行：

```bash
psql -h localhost -U postgres -d cycling_smart -f database\add_slope_fields.sql
```

**预期输出：**
```
ALTER TABLE
ALTER TABLE
CREATE INDEX
CREATE INDEX
```

### 步骤2：导入DEM数据

**前提条件：**
- 确保DEM文件存在：`data\章贡区.tif`
- 需要安装PostGIS工具包（包含raster2pgsql）

**执行命令：**
```bash
raster2pgsql -s 4326 -t 250 -I -C -M -N 32 data\章贡区.tif public dem
```

**参数说明：**
- `-s 4326`：设置坐标系为WGS84
- `-t 250`：设置瓦片大小为250米
- `-I`：导入后立即退出
- `-C`：使用COPY模式（更快）
- `-M`：使用瓦片模式
- `-N 32`：设置NODATA值为32
- `data\章贡区.tif`：DEM文件路径
- `public dem`：目标表名

**如果没有raster2pgsql：**
可以跳过此步骤，直接执行坡度计算（但坡度值将为0）

### 步骤3：计算坡度

```bash
psql -h localhost -U postgres -d cycling_smart -f database\calculate_slope_simple.sql
```

**预期输出：**

## 🔍 验证结果

### 检查字段是否添加成功

```sql
SELECT column_name, data_type
FROM information_schema.columns
WHERE table_name = 'road' 
  AND column_name IN ('elevation_start', 'elevation_end', 'avg_slope', 'max_slope', 'slope_category');
```

### 查看坡度分类统计

```sql
SELECT slope_category, COUNT(*) as count
FROM road
WHERE slope_category IS NOT NULL
GROUP BY slope_category;
```

### 查看示例道路的坡度

```sql
SELECT id, name, road_type, length_km, avg_slope, max_slope, slope_category
FROM road
WHERE avg_slope IS NOT NULL
ORDER BY max_slope DESC
LIMIT 10;
```

## 🐛 常见问题

### Q1: psql命令提示输入密码
**A:** 使用环境变量设置密码
```bash
set PGPASSWORD=123456
psql -h localhost -U postgres -d cycling_smart ...
```

### Q2: 找不到raster2pgsql
**A:** 需要安装PostGIS工具包
- 下载：https://postgis.net/windows_downloads/
- 安装PostGIS Bundle（包含raster2pgsql）

### Q3: DEM文件太大，导入很慢
**A:** 可以调整瓦片大小参数
- 使用更大的瓦片：`-t 500`
- 或先裁剪DEM到项目区域再导入

### Q4: 坡度计算结果为0
**A:** 可能原因：
1. DEM数据未导入
2. 道路数据与DEM区域不重叠
3. 坐标系不匹配

**解决方法：**
```sql
-- 检查DEM表是否存在
SELECT EXISTS (SELECT FROM information_schema.tables WHERE table_name = 'dem');

-- 检查道路高程是否计算
SELECT COUNT(*) FROM road WHERE elevation_start IS NOT NULL;
```
